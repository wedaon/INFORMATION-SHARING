# 마크다운
### 1.개요
* 마크다운은 그냥 글 보기 쉽고 예쁘게 만들기 위해서 쓰는 언어이다.
* 쓰기 위해선 VSCODE안에서 작성하거나 깃허브에서 문서를 만들 때 .md를 뒤에 붙여야 된다.

### 2.사용법
+ 내가 마크다운를 써보면서 자주 쓰는 언어 위주로 설명하도록 하고 더 자세한 내용은 맨 밑의 링크를 참조하면 된다.
2-1.글머리:앞에 #쓰고 한 칸 띄고 쓴다. 쓰는 언어의 크기를 설정한다 보면 된다. #이 1~6개 까지 되고 적을 수록 크기가 크다.
     
  예시)
     
  \# 글머리
  # 글머리
  \### 글머리
  ### 글머리
  \###### 글머리
  ###### 글머리


2-2.인용:글 밑에 살짝 흐리게 추가요소를 더 해주는 언어이다. 이것 또한 >쓰고 한칸 띄고 원하는 말 적으면 된다.   
예시)

\> 날자꾸나 부러진 날개야
> 날자꾸나 부러진 날개야
