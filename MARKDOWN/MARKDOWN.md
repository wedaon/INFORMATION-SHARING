# 마크다운
마크다운에 대하여
--------
### 1.개요
+ 마크다운: 마크다운은 문서를 보기 편하고 깔끔하게 정리하기 위해 쓰는 언어로 주로 깃허브나 VSCODE에서 쓰인다.
>깃허브에서 쓸 때는 문서를 만들 때 뒤에 .md를 붙여야 한다.
+ 여기서는 내가 편하게 자주 사용하는 문법만 정리 해놓았으니 자세한 내용은 맨 밑의 링크를 참조하기 바란다.
*******
### 2.사용법
+ 글머리: 앞에 #를 쓰고 한 칸 띄어서 쓰며 글의 크기를 키울 때 주로 사용하며 1~6개 까지 #을 쓸 수 있고 적을 수록 글의 크기가 커진다.   
  ex)   
<pre><code>
# 글머리
### 글머리
###### 글머리
</code></pre>
출력 예   
# 글머리
### 글머리
###### 글머리   

*****
   
+ 인용 문자: 앞에 >를 쓰고 한 칸 띄어서 쓰며 밑에 흐리게 추가 할 말을 삽입 할 수 있다.   
 ex)   
<pre><code>
> 새는 알에서 나오려고 투쟁한다.
</code></pre>
 출력 예   
> 새는 알에서 나오려고 투쟁한다.

******

+ 글머리 기호: 앞에 *,-,+ 중 하나를 쓰고 한 칸 띄어서 쓰며 •를 앞에 추가 할 수 있다.   
ex)   
<pre><code>
* 글머리 기호
</code></pre>
 출력 예   
- 글머리 기호

*****
* 코드 블럭:맨 앞에 \<pre>\<code>를 쓰고 사이에 문자를 넣은 다음 맨 뒤에 \</code>\</pre>를 넣으면 코드 블럭으로 추가된다.
 > 위에서 보인 것처럼 코드 블럭안에 설명 같은 걸 적어도 문제는 없다.    

ex)   
      
   \<pre>\<code>     
x=int(input("다이아몬드 시작부터 중앙의 길이를 입력해주시오"))   
for h in range(1,x+1):   
 print(" "*(x-h),"*"*(h+h-1))    
for i in range(x-1,0,-1):   
 print(" "*(x-i),"*"*(2*i-1))   
      
  \</code>\</pre>      

 출력 예   
  <pre><code>
x=int(input("다이아몬드 시작부터 중앙의 길이를 입력해주시오"))
for h in range(1,x+1):
 print(" "*(x-h),"*"*(h+h-1))
for i in range(x-1,0,-1):
 print(" "*(x-i),"*"*(2*i-1))

  </code></pre>
   
*****
   
* 수평선: -----, ***, ***** 이것들은 모두 수평선을 만든다
> 그냥 -,*를 3개 이상으로 쓰면 된다. 참고로 ---는 단어 밑에 있으면 부제목이 됨으로 ***를 쓰는 걸 추천 함.

ex)
      
   \----   
   
  출력 예   
  ***   
     
***   
   
+ 링크: [제목]\(link)이런 형태로 적으며 (link, "부가 표시")이렇게도 할 수 있다.
   
 ex)   
 <pre><code>
   [MARKDOWN](https://gist.github.com/ihoneymon/652be052a0727ad59601, "마크다운에 대한 모든 것".)
 </code></pre>

  출력 예   
   [MARKDOWN](https://gist.github.com/ihoneymon/652be052a0727ad59601, " 마크다운에 대한 모든 것")

***
   
* 강조: 약한 강조:\*강조\*, 강한 강조:\*\*강조\*\*, ~\~검열~\~의 형태로 쓰인다.   
   
ex)  
 <pre>
   *강조*, **강한 강조**, ~~검열~~
 </pre>

  출력 예   
  *강조*, **강한 강조**, ~~검열~~

  ***

  * 그 외: 줄바꿈은 문장의 끝에서 띄어쓰기를 3번 이상 하는 것으로 되며 그냥 엔터만 쳐서 넘길시 합쳐진다.
띄어쓰기는 \&nbsp;이걸 쓰면 없어지지 않고 출력 가능하다.
